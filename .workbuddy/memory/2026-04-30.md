## company.html 格式错误问题

### 问题描述
- 公司详情页显示"正在加载中"，无法正常显示数据
- 根本原因：批量添加 founders 数据时产生了 JavaScript 语法错误
- 错误类型：对象缺少闭合括号 `}`、逗号 `,`，导致 JS 解析失败

### 修复方法
1. 从 git 恢复文件：`git checkout company.html`
2. 使用正确的版本（commit `5d4a434`）作为基础
3. JS 语法验证：`node --check _temp_check.js`

### 预防措施
- 批量更新脚本需要严格验证输出格式
- 每次修改后用 `node --check` 验证 JS 语法
- commit 前检查文件行数和基本格式
